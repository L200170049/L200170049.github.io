Perintah Dasar DDL


1. CREATE
Perintah ini digunakan untuk membuat, termasuk diantaranya membuat database baru, tabel baru, view baru, dan kolom.
Bentuk Umum :
CREATE DATABASE nama_database;

2. CREATE
Bentuk Umum :
CREATE TABLE nama_tabel (kolom1 tipe_data(panjang),kolom2 tipe_data(panjang),â€¦kolom_n tipe_data(panjang),
PRIMARY KEY (nama_kolom));

3. ALTER
Perintah ini digunakan untuk mengubah struktur tabel yang telah dibuat. Pekerjaannya mencakup mengganti nama tabel,
menambah kolom, mengubah kolom, menghapus kolom, maupun memberikan atribut pada kolom.
a. Menambahkan Kolom/Field
Bentuk Umum :
ALTER TABLE table_name ADD column_name datatype;

b. Menghapus Kolom/Field
Bentuk Umum :
ALTER TABLE table_name DROP column_name datatype;

4. RENAME
Perintah yang digunakan untuk merubah nama Objek
Mengubah Nama Tabel :
Bentuk Umum :
RENAME TABLE table_name TO new_table name

5. DROP
Perintah ini digunakan untuk menghapus database dan tabel.
a. Menghapus Table, Bentuk Umum :
DROP TABLE nama_tabel;

b. Menghapus Database, Bentuk Umum :
DROP DATABASE nama_database;

6. Truncate
Menghapus semua isi dari suatu table database.
Bentuk Umum :
TRUNCATE TABLE table_name;

7. Desc/Describe
Perintah Untuk melihat struktur table secara detail.
Bentuk Umum :
DESC buku;
